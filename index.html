A little programming language developed from Chapter 11 of Eloquent Javascript.

Syntax

  • Assignment: define([var], [val]). Assigns a value to a variable.
    For example: define(a, 10) assigns the value 10 to variable a

  • Nonlocal Assignment: set([var], [val]). Assigns a value to a variable. If the variable is not found in the local scope, it will traverse each outer scope until the variable is found. If the variable is not defined in any scope, a ReferenceError will be thrown.
    For example: do(define(x, 4), define(setx, fun(val, set(x, val))), setx(50), print(x)) will return 50.

  • Output: print(output). Prints the expression output to the result text area.
    For example: print("Hello world!") outputs the string "Hello world!" to the result text area.

  • User input: input([prompt], [default]). Runs a popup window and returns the entered input. Arguments [prompt] and [default] are optional. [prompt] is a text prompt given to the user. [default] fills out the entry box with default text.

  • Code blocks: do([statement1], ...). Executes the statements in the parentheses separated by commas.
    For example: do(print(5),define(a, 2)) prints "5" to the result text area and assignes the value 2 to variable "a".

  • Operators: [operator]([operands]). Performs the operation [operator] on the operands in parentheses. Valid operators are

    • + Addition
    • - Subtraction
    • * Multiplication
    • / Division
    • == Equality
    • < Less than
    • > Greater than

    For example: +(2,3) performs the operation 2+3 resulting in a value of 5.

  • Functions: fun([arg1], [arg2], ..., [funtion]). A series of named arguments are provided first. The final argument is the function logic.
    For example: define(plusOne, fun(a, +(a, 1))). This assigns the variable plusOne with a function that takes a single argument a and adds 1 to it.

  • If statement: if([test], [true], [false]). Tests condition [test] and executes either [true] or false based on result.
    For example: if(==(4,5),print("equal"),print("not equal")) outputs "not equal".

  • While loop: while([condition], [statement]). Execute [statement] while [condition] is true.
    For example: do(define(a,0),while(<(a,10),do(print(a),define(a,+(a,1))))) prints out a sequence of numbers 0 - 9.

  • Array definition: array([el1], ...). Defines an array of elements.
    For example: do(define(a, array(1, 2, 3)), print(a)) outputs 1,2,3 to the result text area.

  • Array length: length([array]). Returns the length of [array].
    For example: do(define(a,array(1, 2, 3)),print(length(a))) outputs 3 to the result text area.

  • Array element: element([array], [index]). Returns the element at the zero based index position [index].
    For example: do(define(a, array(1, "x", 7)),print(element(a, 1))) outputs "x" to the result text area.
